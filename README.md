# ProTracker

ProTracker is a bullet-journal inspired productivity web app built with vanilla HTML, CSS, and JavaScript.

## Features

- Daily task log with priority and completion tracking
- Habit tracker with streak counting
- Goal/project planner with deadlines
- Notes section with autosave
- Monthly overview calendar with activity dots
- Pomodoro focus timer (focus and break customization)
- Mood/energy wellbeing check-ins with history
- Analytics snapshot
- Data export/import and full reset

## Run locally

Because this is a static app, you can run it with any simple local server:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.
