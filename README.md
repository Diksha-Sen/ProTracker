# ProTracker

ProTracker is a bullet-journal inspired, interactive productivity web app made with vanilla HTML/CSS/JS.

## What it includes

- Daily to-do tracker with priorities
- Sticky section navbar for quick navigation across all journal areas
- Routine trackers with custom frequency in days (habit, skin care, hair wash, cloth washing)
- Sleep tracker
- Mood + anxiety tracker
- Goals and projects tracking
- Monthly / weekly / yearly planner sections
- Book, audio book, movie, and song logs with star ratings
- Automatic media sorting by rating (highest first)
- Monthly activity calendar
- Notes/reflections with autosave
- Progress snapshot analytics
- Larger, roomier card layout for easier reading and planning
- Export/import/reset for backups

## Run locally

```bash
python3 -m http.server 4173
```

Open: `http://localhost:4173`
